# @arkonjs/json-rpc

[![npm version](https://img.shields.io/npm/v/@arkonjs/json-rpc.svg)](https://www.npmjs.com/package/@arkonjs/json-rpc)

This package provides a light framework for implementing a
[JSON-RPC 2.0 API](https://www.jsonrpc.org/specification).
